# \<odg-img>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i odg-img
```

## Usage
```html
<script type="module">
  import 'odg-img/odg-img.js';
</script>

<odg-img></odg-img>
```
